# 第一章　可转债与下修机制：入门与基础

## 1.1 可转债是什么

可转换公司债券，是一种"债券+股票期权"的混合融资工具。简单说，就是发行人向投资者借钱，除了约定还本付息之外，还给了持有人一个额外的权利：在规定期限内，可以按照事先约定好的价格（转股价）把债券换成公司股票。

这个"换股权"是可转债区别于普通债券的核心所在。持有人拥有的，是一张带有期权的借条：最坏的情况是持有至到期、收回本金和利息；最好的情况是正股大涨，以较低的转股价换成股票，享受上涨收益。

一张可转债的价值，可以拆解为以下几个部分：

**纯债价值（债底）** 是把可转债当作普通债券来对待，将未来各期利息和到期本金折现得出的现值。A股可转债的票面利率通常采用"前低后高"的递进结构——第一年约0.3%至0.5%，逐年递增，最后一年约2%至2.5%，加权平均约1.5%至2%。票息偏低是常态，发行人本质上是用较低的利率换来给持有人的那个"转股期权"。债底的意义在于：当正股跌得一塌糊涂、转股毫无价值的时候，可转债价格不会无限下跌，债底是那个兜底的"地板"。

**转股期权价值** 是可转债最核心的吸引力。持有人有权（而非义务）在转股期内将债券转成正股。转股期通常在发行结束6个月后开始，直至到期日前5个交易日。转股价的确定方式通常以公告日前20个交易日均价和前1日均价两者较高者为基准，防止发行人在高点附近"多圈股票"。

**向下修正期权价值** 是发行人的一项主动权——当正股价格持续低迷时，发行人有权主动下调转股价，使原本毫无吸引力的转股条件重新变得诱人。这是本报告的核心研究对象。

**赎回与回售条款** 分别保护发行人和持有人。强赎条款（正股连续大幅高于转股价130%时，发行人可强制赎回）保护发行人不被持有人"永远套牢"；回售条款（进入最后两个计息年度后，正股持续大幅低于转股价，持有人可按面值回售）保护持有人在极端情况下不至于血本无归。

理解可转债的本质，关键要看清这个博弈的底层逻辑：发行人想要的是"永远不还钱"——促成持有人转股，把债务变成股权，永久降低负债率；持有人想要的是在承担有限下行风险的同时，吃到正股上涨的肉。下修条款，正是这场博弈里发行人最重要的主动工具。

## 1.2 下修机制的经济学逻辑

下修的本质不是"让利给持有人"，而是发行人的一次主动权益置换。

对发行人而言，可转债融资的隐含成本其实低于普通债券——持有人愿意接受较低票息，是因为他们获得了那个"免费"的转股期权。如果正股一直表现低迷、持有人最终不转股，发行人到期就得还本付息，这笔"期权费"相当于全部由发行人事后来补。所以，发行人有极强的内生动力促成转股。在正股持续低迷的时候，这种动力转化为下修的冲动。

下修的本质是：把那张已经变得毫无价值的转股期权，以更低的价格重新"激活"——一张深度价外的期权一文不值，而一张接近或处于平价的期权，持有人才愿意考虑行权。

对持有人而言，下修带来的好处也很直接：转股价降低意味着每张债券可以换到更多股票。假设转股价从10元降至7元，每张面值100元的债券可转换股数从10股增至约14.3股，增幅43%。与此同时，下修公告本身往往带动正股短期反弹，且传递了发行人强烈的促转股意愿，持有人安全感也随之上升。

当然，下修不是没有风险。若下修幅度过大，潜在的股本稀释效应会对正股形成压力，甚至抵消转股价值提升带来的收益。这个矛盾，是评估下修性价比时绕不开的核心变量。

## 1.3 下修触发机制详解

下修条款的触发，通常需要同时满足时间条件和空间条件。

**空间条件**方面，主流设置是正股收盘价低于当期转股价的80%或85%。部分2024年后发行的新券，为增强对持有人的吸引力，将触发线提高到了90%，意味着正股只要跌破转股价10%就触发，条款更为宽松。

**时间条件**方面，最常见的是"连续30个交易日中至少15日正股收盘价低于转股价的85%"，也有设置为"连续20个交易日中至少10日"等变种。值得注意的是，触发是一个累积过程——正股某天反弹到触发线以上，并不会"清零"此前已经累积的触发日，只要此前累积的触发日数量达到条款要求，发行人就已经获得了下修权利，后续可以在任何时候选择行使。

**修正幅度的约束**是理解下修空间的关键。多数条款规定：修正后的转股价不得低于最近一期经审计的每股净资产。这一"净资产约束"是刚性上限——如果正股PB已经接近或低于1，下修空间可能极为有限，触发条款的意义大打折扣。此外，单次下修幅度通常不超过20%，多次下修才能累积较大的降幅。

## 1.4 下修决策的内部流程

下修的决策链条，因发行人性质不同而存在显著差异。

民营上市公司的流程相对简洁：管理层提出议案→董事会决议→（需要时）股东大会审批→公告实施。从触发到公告，通常5至30个交易日，取决于管理层的决策效率。

部分发行人会在公告前后配合释放利好（业绩预告、大额订单、回购股票等），形成"下修+利好"的共振效应，这种操作在合规层面完全允许，只是考验发行人的市值管理能力。

国有企业的决策链条明显更长。国有资产的稀释涉及国资监管部门的审批程序，决策周期可能长达数月。实践中，国企即便触发了下修条款，也往往无法快速响应，这是评估国企可转债下修可行性时必须考量的重要约束。

## 1.5 下修后的典型市场路径

下修公告发出后，市场走势通常遵循一个相对可预测的节奏。

公告当日或次日，可转债价格通常上涨1%至5%，反映市场对下修条款的价值重估。随后的3至10个交易日，正股可能出现一定压力，因为部分套利资金会在利好兑现后离场，稀释效应也开始被消化。之后进入"验证期"——下修能否成功，最终取决于正股能否在修正后的转股价上方持续运行。

历史上，约60%至65%的标的在下修公告后30个交易日内实现正收益，胜率虽不算极高，但结合可转债的债底保护，整体风险收益比远优于直接持有正股。

历史数据还揭示了几条值得关注的规律：绝大多数（90%以上）可转债最终以转股或触发强赎告终，真正到期还本付息的案例极为罕见；下修公告频率与市场整体走势高度负相关，熊市或震荡年份（如2018年、2022年）下修数量显著多于牛市年份；曾经下修过的发行人，再次下修的概率明显高于从未下修过的——"下修惯犯"的逻辑值得重视。

# 第二章　下修条款的法律约束与监管边界

## 2.1 法规体系

可转债的发行与下修，涉及多层次法规框架。《公司法》和《证券法》是根本依据；证监会发布的《上市公司证券发行注册管理办法》（2023年修订版）是核心上位制度；两大交易所的《可转换公司债券业务指引》规范具体交易与信息披露；每只可转债的《募集说明书》则是法律合同，具体约定了触发条件、修正幅度和实施程序，持有人据此主张权利。

## 2.2 典型条款文本的法律要点

以一份典型的下修条款为例：

> "在本次发行的可转债存续期间，当公司股票在任意连续30个交易日中至少有15个交易日的收盘价低于当期转股价的85%时，公司董事会有权提出转股价格向下修正方案并提交公司股东大会审议表决。修正后的转股价格应不低于该次股东大会召开日前20个交易日公司股票交易均价和前1交易日公司股票交易均价之间的较高者，且不低于最近一期经审计的每股净资产值。"

从这段文字中，几个法律要点值得特别关注：

**下修是权利，不是义务。** 条款赋予发行人"提出修正方案"的权利，而非强制要求。触发条件满足后，发行人有权选择行使，也有权搁置不动。持有人无法强迫发行人下修，只能对发行人的行为方式形成预期判断。

**股东大会表决是必要门槛。** 下修方案需经出席股东大会的股东所持表决权的三分之二以上通过。控股股东的意见基本上决定了结果，大股东的持股比例、稀释敏感度和对资本结构的态度，直接影响下修能否通过。

**净资产下限是硬约束。** 转股价的下修空间，最终受制于"每股净资产"这条线。PB接近1的标的，名义上触发了下修条款，实际上几乎没有可调节的空间。

**时间和幅度双重约束。** 部分条款规定每计息年度至多下修一次、单次幅度不超过初始转股价的30%，将转股价从10元下修至4元，理论上需要至少两到三次、跨越多个计息年度。

## 2.3 监管的特殊要求

触发下修条款后，发行人须在两个交易日内发布公告，说明是否拟下修；若不下修，同样须公告说明原因。

内幕信息管理方面，下修决策在正式公告前属于内幕信息。参与决策的人员不得在敏感期内买卖相关可转债或正股。近年来内幕交易执法力度加强，这是发行人决策时必须严守的合规红线。

大股东减持方面，部分情形下大股东减持正股需遵守减持新规，减持行为本身可能对正股价格形成压力，形成负反馈。这一问题在持股比例集中的可转债标的中尤需关注。

## 2.4 国有企业的特殊约束

国有控股上市公司的可转债下修，还叠加了国资监管规则的约束。部分地方国资监管机构对下修行为要求进行资产评估备案，审批层级的提高意味着更长的决策周期和更大的不确定性。国有资产保值增值考核机制也使管理层对稀释效应存在天然的顾虑。

实践中，国有企业可转债的下修公告频率显著低于民企，从触发到实施的时间间隔也普遍更长。在评估国企可转债的下修可行性时，这些制度约束不能视而不见。

## 2.5 条款理解中的几处灰色地带

即便在文字约定清晰的情况下，实践中仍存在若干值得关注的理解分歧：

"任意连续30个交易日"的起算时点如何确认？是否允许跨年度连续计算？不同募集说明书的约定存在差异。"就高原则"的适用有时会造成修正空间的意外扩大或收窄。已下修后重新触发的起算基准，也因条款设计不同而有所差异。这些技术细节，对于精确评估具体标的的下修空间具有实质性影响，需要逐条款仔细阅读。

# 第三章　本周宏观环境与A股市场复盘

## 3.1 宏观背景

2026年3月，全球资本市场处于一个充满不确定性的节点。几条主线交织：

**地缘政治风险重燃。** 中东局势年初以来持续紧张，美以联合军事行动与伊朗的激烈回应，将霍尔木兹海峡推向封锁边缘。这条通道承担着全球约20%的石油贸易，封锁威胁直接冲击全球油价、抬升避险情绪，VIX指数飙升，全球资金向美元和黄金流动。对中国经济的影响是双向的——进口成本上升对炼化、化工、航空等行业形成压力，但也进一步强化了新能源替代需求的逻辑。

**全球经济进入"弱复苏+高波动"阶段。** 美国经济出现增长放缓迹象，但就业市场依然韧性十足，通胀粘性超预期，美联储在2025年降息两次后，于2026年初暂停降息，政策利率维持在4.25%至4.50%区间。外部需求的分化对中国出口形成结构性影响——欧美需求放缓，"一带一路"沿线相对稳健。

**国内经济保持5%左右增长。** 2026年是"十五五"开局年，政府工作报告延续了扩大内需、科技创新、产业升级的政策主线。1至2月外贸数据出口同比增长约7%，大超预期；消费方面社零同比增长约5.5%，服务消费明显快于商品消费。战略性新兴产业（新能源汽车、半导体、AI、储能）持续获得政策倾斜。

两会期间的政策基调对资本市场偏友好：证监会强调提升上市公司质量、完善退市制度、健全中长期资金入市机制，对可转债市场的整体信用环境和长期发展均构成支撑。

## 3.2 本周A股走势复盘

2026年3月22日至27日，是2025年以来A股波动最为剧烈的一周。

**周一（3月23日）：黑色星期一。** 受中东地缘冲突骤然升级冲击，A股大幅低开后跌幅持续扩大。上证指数盘中一度下探至3794.68点，尾盘险守3800点关口，最终收于3813.28点，单日暴跌3.63%；深证成指下跌3.76%，创业板指下跌3.49%。全市场超过6000只个股下跌，跌停家数134只，主板成交额放量至2.44万亿元，主力资金净流出1254亿元。对可转债市场而言，这一天的意义尤为特殊——大量中小盘正股的跌幅将一批可转债的转股价偏离度在一日之内从"安全区"推入"触发观察区"，也在历史低位为逆向投资者创造了买点。

**周二（3月24日）：高开震荡。** 地缘局势出现阶段性缓和信号，市场情绪有所回暖，三大指数高开但未能持续，全天整体震荡回落，成交额萎缩至约2.08万亿元，较周一大幅缩量，空方力量趋于衰竭。

**周三（3月25日）：全线大涨。** 美伊谈判取得进展的预期大幅升温，市场风险偏好显著改善。上证指数收于3931.83点，大涨1.30%，成功收复3900点整数关口；全市场超4500只个股上涨，成交额放大至约2.18万亿元，资金开始试探性入场。这一天对可转债市场是个关键节点——部分在周一触发下修条款的正股，在周三反弹后重新回到了"触发观察期"，发行人的策略选择变得微妙。

## 3.3 行业表现与板块分化

本周各板块走势高度分化，这一分化直接映射到可转债正股的表现上。

**相对强势的板块：** 有色金属（尤其贵金属）受金价大涨驱动，全周表现居前；医药板块展现防御属性，在动荡中相对抗跌；电力设备（特高压、智能电网方向）受"十五五"电网投资加速预期提振，表现活跃；锂电/储能在周五集中爆发——1至2月国内储能电芯产量同比增长91%、3月排产环比增长16.5%的数据强劲，提振了整个赛道的信心。

**承压明显的板块：** AI与算力、半导体材料及设备是本周受冲击最大的板块，2025年的强势积累了较大估值压力，外围冲击成为套利离场的导火索；光伏产业链继续承受产能过剩与出口压力的双重拖累；消费电子表现分化，智能手机零部件有所承压。

**资金面：** 全周主力资金合计净流出约1320亿元，其中周一单日净流出1254亿元。北向资金成交日均约2445亿元，约占两市总成交额13%，未出现大规模异常流出，外资对A股中长期信心相对稳定。

# 第四章　本周波动对下修格局的影响

## 4.1 正股偏离度的脉冲式变化

本周最直接的效应，是大量中小盘可转债的转股价偏离度在周一急跌中发生了骤变。

以一个具体情形来说明：某可转债正股在周一前收盘价8.50元，转股价8.00元，处于正股略高于转股价的"安全状态"，距下修触发线（转股价的85%=6.80元）尚有25%空间。周一正股跌幅7%，跌至7.91元，仍在安全区。但对于正股原本就处于转股价附近的标的，单日7%至10%的跌幅足以将其直接推入触发观察区，甚至当日就跌穿触发线。

典型的触发条件是"连续20个交易日中至少10日正股收盘价低于触发线"。周一的急跌，相当于在1天内贡献了1个"触发日"，若正股随后数周持续低迷，触发下修的概率将急剧上升。

周三的反弹改变了部分标的的处境：那些在周一仅刚刚触及触发边缘的标的，反弹后重新远离了触发线；但对于周一已经深度跌穿触发线的标的（如正股直接跌至6.00元），即便周三反弹至6.50元，依然处于触发线以下，触发状态并未解除。

从公告节奏看，本周触发下修条款的标的，其正式下修公告可能集中出现在4月上旬——发行人需要1至2周的内部审批流程，且精明的发行人往往会等待市场情绪修复、正股技术形态改善后再择机公告，以获得更好的市场反应。3月25日至27日期间的正股反弹，实际上为部分标的提供了理想的下修公告窗口。

## 4.2 发行人动机的分化

本周波动对不同类型发行人下修动机的分化效应尤为明显。

**民营中小企业是压力最大的群体。** 这一群体占A股存续可转债数量的七成以上，面临的是多维度压力叠加：民营企业的信息透明度较低、股东结构较分散，恐慌时更容易成为抛售对象；融资渠道相对有限，到期兑付的现金流压力更为真实；但与此同时，其决策自主性最强，促转股的战略意愿也最为迫切。对于基本面稳健、PB充裕的优质民企而言，本周的急跌是难得的窗口——以更低的稀释成本推进转股，或者抱着"跌了正好出手"的心态启动下修程序。

**国有企业是动机最弱的群体。** 国资审批链条长、保值增值考核压力、管理层对短期股价的不敏感，共同构成了国企下修的制度性阻力。但值得关注的是，部分地方国企（尤其是城投平台转型类公司）在近期城投融资政策收紧背景下，再融资压力并不亚于民企，其实际推动下修的紧迫性不能简单类比央企或大型地方国企。

**银行等金融机构的可转债**受到监管部门对资本充足率和股权稀释的额外审慎监管，下修的实际操作空间和意愿受到更大约束，可以视为特殊类别单独看待。

## 4.3 持有人行为的周期演绎

本周也清晰呈现了可转债持有人行为的典型周期。

周一的恐慌抛售，与持有人的总体组合回撤控制压力密切相关——当股票大跌时，持有可转债的账户同样面临回撤，被动卖出可转债以补充保证金或降低风险敞口，是常见操作。这种非理性的恐慌性抛售造成了两类错误定价：基本面良好但被误杀的标的跌幅超过基本面应有水平，流动性较差的小盘可转债因市场深度不足而价格冲击成本更高。

周二的缩量观望，周三的试探性加仓（部分激进资金逢低布局超跌反弹），周四至周五的理性分化——持有人行为逐步从情绪主导回归到基本面判断，经历了完整的"恐慌→观望→试探→理性"周期。

对于下修博弈而言，最有价值的买点恰恰在恐慌期——此时价格偏离基本面最远，赔率最高。但投资者面临的难点，是在恐慌中需要有足够的确定性判断来逆向行动，而不是事后追涨。

# 第五章　下修可行性多维评估框架

## 5.1 转股价偏离度

偏离度是评估下修可行性最核心的量化起点，计算方式为：

> 偏离度 = （正股当前价 - 转股价）/ 转股价 × 100%

更精确的指标是"距触发线的空间"：

> 距触发线空间 = （正股当前价 - 触发线价格）/ 触发线价格 × 100%
> 触发线价格 = 转股价 × 触发比例（通常80%或85%）

实务中，常用以下区间来判断下修压力等级：

| 偏离度区间 | 处境描述 | 下修压力 |
|-----------|---------|---------|
| > 0% | 正股高于转股价，处于价内 | 极低 |
| -10% 至 0% | 正股略低于转股价 | 低 |
| -20% 至 -10% | 中度偏离，部分标的进入触发观察 | 中等 |
| -30% 至 -20% | 显著偏离，多数已触发或逼近触发线 | 较高 |
| < -30% | 深度偏离，条款大概率已触发 | 极高 |

有一点很容易被忽视：对于PB接近或低于1的标的，偏离度再大也没有实际意义。净资产约束卡死了下修空间的上限——某可转债转股价10元，正股跌至7元，偏离度-30%，看似很深；但若每股净资产为9.50元，修正后的转股价最低也不能突破9.50元，几乎没有任何实质性空间。这种"名义偏离高、实际空间极小"的情形，在国有企业和重资产行业（煤炭、钢铁、部分房地产标的）中尤为常见。

综合偏离度、PB空间、发行人性质、剩余期限、历史行为，可以构建一套评分框架：偏离度基础分（深度偏离80-100分，以此类推），PB空间加减分（PB>3倍加10分，PB<1倍扣20分），民企加10分/央企扣10分，剩余期限2年内加15分，有下修历史加10分。综合分80分以上的，属于高概率下修候选。

## 5.2 可转债价格区间

可转债价格是债性与股性博弈的结果，不同价格区间的下修博弈价值截然不同。

**100元以下**：债性主导。此时可转债的定价主要反映其持有到期的期望收益，正股的涨跌对价格的影响相对有限。持有人的关注重心是发行人的偿付能力，而非转股收益。下修在这一区间的价值较弱——如果可转债已经跌破面值，往往意味着信用风险已经被市场定价，下修解决不了根本问题。

**100至110元**：这是下修博弈的主战场。面值附近的可转债，债底提供了充分的下行保护（持有到期的底线极为清晰），同时正股偏离度又足够大（否则价格早已远高于面值），下修条款一旦兑现将带来可观收益。本周周一急跌后大量标的落入这一区间，周三反弹带动部分标的重新站上110元，但仍有相当数量在100至105元附近徘徊，继续提供良好的风险收益比。

**110至130元**：股性渐强，下修的必要性和博弈价值均有所下降。这一区间的可转债定价更接近"带债底保护的股票"，其价格走势与正股的关联度已经相当高。

**130元以上**：强赎边界区，与下修博弈基本无关。持有人此时应重点关注的是强赎条款是否触发、正股能否维持在触发线上方。

## 5.3 剩余期限与余额规模

**剩余期限**是下修紧迫性最直接的时间指标。

距到期超过4年：从容期，发行人通常选择观望。距到期3至4年：开始进入议程，但紧迫感有限。距到期2至3年：甜蜜期。这个时间窗口最为理想——既有足够的时间等待正股反弹，又有明显的紧迫感推动发行人主动行动。距到期1至2年：关键期，促转股压力急剧上升，下修往往处于"箭在弦上"状态。距到期1年以内：倒计时期。此时下修的边际效果递减，即便大幅降低转股价，也很难在一年内让正股大幅上涨到足以吸引转股的程度，持有人更倾向于锁定回售收益。

**余额规模**影响稀释效应和持有人结构。

小规模可转债（余额3亿元以下）：稀释压力有限，即便全额转股，新增股本通常在总股本的5%以内；持有人结构较集中，发行人与主要持有人的沟通成本较低；总体看，小规模标的的下修推进顺畅度最高。

中等规模（3至8亿元）：稀释效应中性，下修推进阻力中等。

大规模可转债（超过8亿元）：稀释压力显著。以一只余额20亿元、初始转股价10元的可转债为例，全额转股意味着新增2亿股；若通过下修将转股价降至7元，转股数量增至约2.86亿股，新增约8600万股。这一规模的潜在供给，会对正股形成持续的价格压力，是评估大规模可转债下修性价比时不能回避的现实约束。

将剩余期限与余额规模交叉分析，高下修紧迫性的标的集中在"剩余2至3年+中等规模"或"剩余1至2年+中小规模"的区间。本周急跌后，一批原本处于"关注期"的标的快速进入"紧迫期"，是值得重点追踪的群体。

## 5.4 溢价率与折价标的

溢价率的计算公式为：

> 溢价率 = （可转债价格 - 转股价值）/ 转股价值 × 100%
> 转股价值 = 正股当前价 / 转股价 × 100

溢价率的本质，是持有人为"持有等待"的权利所支付的期权费。在正常市场中，可转债的溢价率是正数——持有人愿意付出一定的溢价来保留转股的灵活性。

本周急跌对高溢价标的的冲击体现在两个层面：正股下跌直接压缩了转股价值，而可转债由于债底支撑跌幅相对有限，结果溢价率反而被动抬高；同时，部分高溢价标的在恐慌抛售中成为优先减持对象，溢价保护反而失效。

折价（溢价率为负数）在正常市场中极为罕见，套利机制通常会在数天内抹平价差。本周周一的恐慌性抛售，由于流动性枯竭、转股后抛压恐慌、机构风控顺周期强化等原因，造成了一批标的的折价。这类被流动性踩踏"打出"的折价，是价值投资者的机会——一旦流动性恢复、情绪修复，折价迅速收窄。

折价并不总是机会，需要区分几种情形：信用风险折价（市场对偿付能力的担忧）是预警而非机会；流动性陷阱（小盘标的、买盘几乎为零，折价是"假折价"，入场成本极高）需要特别警惕；真正值得关注的，是基本面扎实的发行人在恐慌中被错误定价的折价标的。

折价与下修的交叉点也值得关注：折价意味着持有人不愿转股，这本身会激发发行人的下修动机——"溢价率甚至为负，说明转股价值远低于债券价值，如果不下修、持有人集体选择不转股，我到期得还出一大笔钱"。折价标的一旦叠加下修公告，收益空间可能相当可观。

## 5.5 持有人结构

十大持有人信息是可转债最有价值的公开信息之一，通过分析持有人结构，可以大致判断下修博弈的真实格局。

持有人集中度高（前十大持有人合计持有60%以上）意味着发行人可以与少数几个大户形成高效的非正式沟通，下修前的"预期引导"工作更易推进，下修公告后持有人转股的意愿也更易形成合力。

发行人大股东或关联方出现在十大持有人名单里，通常是强烈的积极信号——大股东用自己的钱押注这只可转债，一旦下修成功、正股上涨，不仅债券收益落袋，还可以通过转股提升持股比例，是双重收益的最优路径。

知名可转债私募或公募基金重仓，往往意味着该标的已经被专业资金深度研究过。私募对条款博弈的理解通常更为深入，其入场信号值得重视。

本周的市场波动提供了一个观察持有人行为的窗口：绝对收益导向型机构（保险、养老金）在急跌中率先减仓，专注于可转债策略的私募则趁势逆向加仓。这种机构行为的分化本身，就是下修博弈机会的重要信号来源。

# 第六章　行业图谱：分板块下修可行性梳理

不同行业的可转债，因PB水平、正股波动性、行业景气度、所有制结构的差异，下修可行性的格局也大相径庭。

## 6.1 科技与半导体：弹性最大，博弈最激烈

科技行业PB普遍在3至6倍之间，AI和半导体赛道部分标的PB高达10倍以上，下修空间在技术上极为充裕。本周外围冲击令这一板块首当其冲，AI概念、半导体设备、CPO等热门赛道周一普遍跌幅超10%，下修触发条件大量满足。

下修可行性在全市场中处于最高区间：触发条件满足，PB空间充裕，民企为主、决策链条短，不少标的还面临2至3年的到期压力。这是本周下修博弈机会最为密集的板块。

但有一个风险不能忽视：部分科技类标的即便下修降低了转股价，正股估值依然偏高，持有人转股的吸引力仍然不足。筛选基本面能够支撑估值的标的，是参与科技类可转债下修博弈的前提。

## 6.2 新能源与锂电：内部分化，精选个股

储能和锂电是本周后半段最强的板块，基本面数据（储能电芯产量同比增长91%、3月排产环比+16.5%）超预期，为正股提供了扎实的基本面支撑。这类标的的下修压力相对较小——正股基本面强劲，发行人对"不下修、等正股自然上涨"的逻辑更有信心。

光伏板块则相反：产能过剩问题尚未根本性缓解，出口面临贸易壁垒，内需增速边际放缓，持续承压。光伏类可转债的下修触发标的较多，但正股基本面的复苏节奏不明朗，下修后的转股吸引力存疑，需要甄别发行人的具体偿债能力。

## 6.3 医药与医疗器械：防御属性+条款博弈并存

医药板块本周展现出明显的防御属性，正股跌幅相对可控。医药类可转债的持有人结构中机构占比通常较高，条款博弈的理解更为成熟。这一板块的下修可行性差异较大，核心变量是发行人的细分赛道（创新药vs.仿制药，高端器械vs.中低端耗材）和商业化进展，需要个股层面深度研究。

## 6.4 有色金属：PB高但下修动力弱

贵金属板块本周受金价大涨驱动，正股基本面强劲、PB较高，下修空间充裕。但这恰恰意味着发行人没有下修的紧迫感——正股本身就在涨，到期压力可控，主动下修只会徒增稀释成本。这一板块的下修博弈机会有限。

## 6.5 银行与国有金融：特殊约束、特殊对待

银行可转债受监管部门对资本充足率和股权稀释的额外审慎监管，下修规则与普通上市公司存在显著差异，实际操作空间和意愿均受到更大约束，建议单独评估、不与其他类型混同分析。

# 第七章　临近到期标的：最后的博弈窗口

## 7.1 临近到期的特殊性

进入到期前最后两年的可转债，进入了生命周期中博弈最激烈的阶段。这一阶段之所以特殊，在于持有人的退出路径从"等待转股"扩展到了"转股或回售"两条路，而发行人则面临"既无法轻易还钱，又难以促转股"的双重困境。

以2026年年初的市场情况来看，进入到期前两年的可转债约有80只，总余额约600亿元，主要是2021年至2022年牛市发行高峰期的那批产品陆续进入晚期。

## 7.2 临近到期的博弈逻辑

发行人在这一阶段的处境是最被动的。可能的应对策略有几种：最后关头大幅下修——一次性下修20%至30%，甚至更多，以激进姿态吸引持有人转股；额外提高票面利率，用更多利息补偿来挽留持有人；在二级市场回购可转债，缩小实际兑付规模；通过资产重组或业绩释放提振正股，为持有人创造转股动力。

持有人在这一阶段的理性计算也很清晰：如果正股价格加上累计债息的总收益，高于持有到期能获得的本息和（面值+累计利息），则继续持有等待转股；否则就回售。以面值100元、票息两年合计约4元、转股价8元、正股价5元为例：转股价值约62.5元（100/8×5），加上利息4元合计约66.5元，显著低于面值回售的104元，持有人自然选择回售。发行人知道这笔账，所以有强烈动机在到期前有所作为。

## 7.3 本周临近到期标的的关注点

本周周一的急跌，令部分临近到期的可转债价格跌至95至98元区间，显著低于100元的回售触发线。这反而吸引了部分理性资金的关注：以95元买入，持有到回售可获约104元，年化收益率约4.7%，高于同期银行存款，且下行空间极为有限（即便不触发回售，到期兑付也有保障）。

在约80只临近到期的标的中，民营企业约占50只，其下修决策自主性更强，"临门一脚"式的大幅下修更有可能出现，博弈价值也更为突出。国有企业则更倾向于"老老实实还钱"，除非有特殊的市值管理或再融资需要。

# 第八章　买卖时机研判与风险红线

## 8.1 买点研判

**触发后、公告前的模糊期。** 下修触发条件刚刚满足、但发行人尚未发出公告的阶段，是赔率最高的埋伏窗口。此时市场尚未充分反映下修预期，价格仍处低位，一旦公告发出价格迅速上涨。难点在于判断哪些标的即将触发，需要结合正股走势、条款设计和大股东行为综合推断。

**市场恐慌期的逆向布局。** 本周周一的急跌创造了大量被错杀的低价标的。历史经验表明，在市场整体恐慌阶段买入可转债，其后3个月内的胜率和赔率均显著高于其他阶段——条件是要排除掉信用风险标的，以免踩入价值陷阱。

**业绩公告与下修公告的共振。** 发行人在靓丽业绩公告后宣布下修，市场反应通常最佳。本周至下周正值年报和一季报密集披露期，业绩超预期+下修公告的双重催化窗口值得重点关注。

**下修公告后的短暂回调。** 部分情况下，公告后正股因稀释预期短暂承压，可转债价格在短暂拉升后出现回调。若正股基本面依然稳健、稀释幅度在预期之内，这类回调往往是比公告当日更理想的买点。

## 8.2 卖点研判

**公告后的情绪高潮（公告后2至5个交易日）。** 此阶段市场情绪达到高潮，预期已充分兑现。风险厌恶型投资者应在这一阶段分批减仓。

**正股再次逼近触发线。** 下修完成后，若正股再度持续下跌并逼近修正后的触发线，说明第一次下修的效果未能有效改善正股处境。此时需重新评估：继续持有还是止损离场，判断的核心是正股基本面是否已经根本性恶化。

**溢价率回归正常区间。** 下修完成后，溢价率若从低位（10%以下）回升至正常水平（20%至25%以上），可转债的进攻性已大幅减弱，这是从"下修博弈标的"切换到"均衡配置标的"的合适时机。

## 8.3 风险红线

**信用风险是最重要的红线。** 发行人已出现债务违约或技术性违约的，下修博弈逻辑基础荡然无存；经营现金流持续恶化、偿债能力存疑的，即便下修成功、转股后持有的正股也不令人安心；外部融资渠道受损、信用评级被大幅下调的，下修只是拖延问题而非解决问题。

**大规模下修的稀释风险。** 对于余额超过10亿元的可转债，一次性深度下修意味着潜在的大规模新增股票供给，在市场整体弱势背景下，这一抛压可能令持有人"转股即被套"，需要仔细权衡稀释成本和转股收益之间的平衡。

**流动性风险。** 余额极小（低于1亿元）的超小盘可转债，买入和卖出时均面临较高的价格冲击成本，这部分流动性成本必须从预期收益中扣除。正股长期停牌或交易极不活跃的标的，定价不确定性极大，不宜参与下修博弈。

**监管与合规风险。** 发行人正在接受监管立案调查的，重大事项推进可能受限，甚至面临退市风险；大股东质押比例过高的，正股下跌引发的补保证金压力可能令大股东失去推进下修的资源和动力。

# 附录一：常用术语

| 术语 | 含义 |
|------|------|
| 转股价 | 债券转换为每股正股的价格，是可转债定价的核心参数 |
| 下修条款 | 转股价格向下修正条款，发行人在正股持续低于转股价一定幅度后可行使 |
| 触发条件 | 连续N个交易日正股收盘价低于转股价的X%，满足该条件后发行人获得下修权利 |
| 净资产约束 | 修正后的转股价不得低于最近一期每股净资产，是下修空间的硬性上限 |
| 溢价率 | 可转债价格相对于转股价值的溢价程度，反映期权时间价值和情绪因素 |
| 债底（纯债价值） | 将可转债视为纯债券时的折现价值，提供价格下行的理论支撑 |
| PB | 市净率，正股股价与每股净资产之比，决定下修的实际空间上限 |
| 偏离度 | 正股价格相对于转股价的偏离程度，是触发下修的最核心量化指标 |
| 强赎条款 | 正股连续大幅高于转股价130%时，发行人可强制赎回可转债 |
| 回售条款 | 进入最后两个计息年度后，正股持续低迷时持有人可按面值回售给发行人 |
| 促转股 | 发行人通过下修、利好配合、市值管理等手段推动持有人转股 |
| 折价 | 可转债价格低于转股价值，溢价率为负，正常市场中极为罕见 |
